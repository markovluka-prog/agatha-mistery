# Описание

Фан-сайт про Агату Мистери. Персонажи, викторины, интерактивная карта, фанфики, иллюстрации, игра.

---

## Стек

- HTML + CSS + JS (статический сайт, GitHub Pages)
- **Supabase** — хранение отзывов, фанфиков, иллюстраций
- **OpenLayers** — интерактивная карта мира
- **i18n** — своя система локализации (RU / EN), через data-i18n атрибуты
- **node_modules/@supabase/supabase-js** v2.95.3

## Страницы

### Главная (index.html)
- Навигация: Главная / Карта мира / Викторины / Персонажи / О нас / кнопка RU (язык)
- Hero: «Добро пожаловать в мир Агаты Мистери!» + описание + кнопки «Исследовать карту» и «Пройти викторину»
- Секция отзывов: список отзывов (из LocalStorage) + форма (имя 50 символов, текст 500 символов, счётчик)

### Персонажи (pages/characters.html)
Данные в JSON прямо в HTML (`<script id="characters-data" type="application/json">`):

| # | Имя | Возраст | Роль |
|---|-----|---------|------|
| 1 | Агата Мистери | 12 лет | Главная героиня, гениальная сыщица, фотографическая память |
| 2 | Ларри Мистери | 14 лет | Двоюродный брат, начинающий писатель, документирует расследования |
| 3 | Ватсон (кот) | — | Сибирский кот с голубыми глазами, верный спутник |
| 4 | Мистер Кент | — | Дворецкий семьи, безупречно вежлив, тайно гордится Агатой |
| 5 | Чандлер Мистери | — | Отец Агаты, владелец детективной школы Eye International |

- Творческая мастерская: кнопки «Отправить иллюстрацию» и «Отправить фанфик»
- Список фанфиков с поиском (загружается динамически)

### Карта мира (pages/map.html)
- OpenLayers карта с попапами по клику на локацию
- Сайдбар со списком локаций (grid)
- Секция иллюстраций фанатов с поиском + кнопка отправить
- Заголовок: «От Лондона до Токио — здесь собраны главные места, где Агата раскрывала загадки»

### Викторины (pages/quiz.html)
Данные в JSON в HTML (`<script id="quizzes-data" type="application/json">`):

**Викторина 1: «Насколько хорошо ты знаешь Агату?»** — 5 вопросов
1. Как должна выглядеть ваша Аватарка? (выбор изображения: Агата / Ларри)
2. Какой особенностью обладает Агата? → Фотографическая память
3. Как зовут кота Агаты? → Ватсон
4. Кем работает отец Агаты? → Владелец детективной школы
5. Кем приходится Ларри Агате? → Двоюродный брат

**Викторина 2: «Приключения по всему миру»** — 4 вопроса
1. В каком городе находится дом семьи Мистери? → Лондон
2. Где Агата расследовала тайну исчезнувшей картины? → В Лувре, Париж
3. В какой стране Агата разгадала проклятие фараона? → Египет
4. Где произошла история с похищенной маской на карнавале? → Венеция

### О нас (pages/about.html)
- Аккордеон (CSS-анимации, с иконкой ▼)
- Контент грузится динамически (Supabase или i18n)
- Сейчас показывает «Загрузка...»
