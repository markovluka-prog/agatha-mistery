# Product Audit

## Что делает сайт уже хорошо
- Есть несколько видов контента: карта, квизы, персонажи, фанфики, иллюстрации.
- Есть международность: RU/EN.
- Есть пользовательский вклад: отзывы, фанфики, иллюстрации.
- Есть глубокий материал по книгам и локациям.

## Чего не хватало до исправлений
- Главная страница почти не показывала глубину сайта: пользователь видел только hero и отзывы.
- Не было явного сценария "с чего начать новому фанату".
- Не было живых превью персонажей, квизов и локаций на главной.
- Творческая мастерская была спрятана слишком глубоко.
- Польза сайта раскрывалась только после нескольких переходов.

## Что стоит сделать дальше
- Добавить отдельный раздел "порядок чтения книг".
- Сделать "карточку книги" с обложкой, локациями, героями и квизами по каждой книге.
- Добавить страницу "новичку во вселенной" с коротким маршрутом по сайту.
- Сделать подборки: "лучшие локации", "самые сложные квизы", "редкие персонажи".
- Добавить избранное или историю просмотренного.
- Добавить блок "новое на сайте" на главную.
- Добавить больше community-механик: голосование за любимую книгу, персонажа, лучшую фан-работу.

## Что исправлено сейчас
- На главную добавлены блоки быстрого старта.
- На главную добавлены живые превью персонажей, викторин и локаций.
- Творческая мастерская вынесена в заметный CTA-блок.
